Customer Experience Analytics Platform

📌 Description
The Customer Experience Analytics Platform is a machine learning-based project that analyzes customer feedback and reviews to determine sentiment (Positive, Negative, Neutral). 
It helps businesses understand customer opinions and improve their services.

---

🚀 Features

- Sentiment Analysis (Positive / Negative / Neutral)
- User input text analysis
- CSV file support for bulk reviews
- Simple and interactive interface
- Fast prediction using trained model

---

🛠️ Technologies Used

- Python
- Machine Learning (NLP)
- Flask (for web application)
- Pandas & NumPy
- Scikit-learn

⚙️ Installation & Setup
1. Clone the repository:

git clone https://github.com/Arpit-aiml/Customer_Experience_Platform.git

2. Navigate to project folder:
cd Customer_Experience_Platform

3. Install dependencies:
pip install -r requirements.txt

---

▶️ Usage

Run the application:
streamlit run app.py

Then open in browser:
 Local URL: http://localhost:8501
 Network URL: http://10.230.130.136:8501

---

📊 Dataset
The project uses a dataset of customer reviews in CSV format for training and testing the sentiment analysis model.
